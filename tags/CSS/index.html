SASS/SCSS - Basic Tutorial

Jul 02, 2016 in Web Development

Install

  • Install sass
    gem install sass

  • Install compass (manage the projects)
    gem install compass

  • Create a project
    compass create [project_name]

    If you want to watch your code, compass watch [project_name]

  • Setting
    Modify the file, config.rb.

    If you don’t want to comment in your css files, line_comments = false.

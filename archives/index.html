Shutdown with Arduino

Jul 27, 2016 in Arduino

Intro

One day, my girl friend said she require a thing that can help her turn off her computer automatically when she goes to the bed. So, I try to use Arduino with a photoresistance to solve her problem.

Arduino

  1. Check References 2(http://coopermaa2nd.blogspot.tw/2010/12/arduino-lab8-led.html).
  2. Write the value of the photoresistance to the serial.

Python

  1. install python (I use 3.4.1.)
  2. install pyserial and py2exe, run python -m pip install -U pyserial and python -m pip install -U py2exe

    If using python 3.5, py2exe would throw an error. Check this.

  4. run python inst.py install and puthon inst.py py2exe

Execute

  1. USB connect the Arduino and PC
  2. Open the execute file test.exe

Change Your Terminal - iTerm2

Jun 28, 2016 in iTerm

Download Zsh

$ curl -L https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh | sh

If you want to change shell: chsh -s [shell] [username]
Get all the shells: cat /etc/shells

You may have problems that some command like nvm worked on bash, but now, it doesn’t. Because you are in zsh. So, you need to copy ~/.bashrc and paste it on the end of the file, ~/.zshrc

Setting Themes: Powerline

  • Download Theme

    Other themes

  • Edit Theme Settings

    ZSH_THEME="powerline"

  • Save and Restart

  • Download Fonts of Powerline

  • Settings

    iTerm->Preference->Profile->Text->Change Font (include “Regular Font” and “Non-ASCII Font”)

Install zsh-completions

  • brew install zsh-completions

  • edit .zshrc

  • rebuild
    rm -f ~/.zcompdump; compinit

Setting Plugin

  • Default plugins are in ~/.oh-my-zsh/plugins
  • In .zshrc, edit plugins=(git)

Sublime-Shortcut Code Folding

Jun 25, 2016 in Sublime

Shortcut of code folding in Sublime

Folding Selected Text

Select the text you want to fold, and hit the shortcut Command + Option + [. Hit Command + Option + [ to unfold.

If no selected text, it would automatically fold or unfold where the cursor is.

Folding All Text By Level

If level is 2, hit Command + K and Command + 2. All text would fold 2nd level text.

So, when level is 0 (Command + K and Command + 0), it would unfold.

Folding Tag Attributes (All Text)

In html files, hit Command + K and Command + T. All text would fold all text by tag attributes.

Hit Command + K and Command + 0. It would unfold all text.

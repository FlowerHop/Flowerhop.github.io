Sublime Code Folding Shortcut

Jun 25, 2016 in Sublime

Shortcut of code folding in Sublime

Folding Selected Text

Select the text you want to fold, and hit the shortcut Command + Option + [. Hit Command + Option + [ to unfold.

If no selected text, it would automatically fold or unfold where the cursor is.

Folding All Text By Level

If level is 2, hit Command + K and Command + 2. All text would fold 2nd level text.

So, when level is 0 (Command + K and Command + 0), it would unfold.

Folding Tag Attributes (All Text)

In html files, hit Command + K and Command + T. All text would fold all text by tag attributes.

Hit Command + K and Command + 0. It would unfold all text.

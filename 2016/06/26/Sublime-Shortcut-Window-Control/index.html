Sublime-Shortcut Window Control

Jun 26, 2016 in Sublime

Shortcut of window control in Sublime

Full Screen

Hit Command + Control + (Shift) + F.

Split N Windows

Hit Command + Option + N(N means number. For example N=3, hit Command + Option + 3).

Working Directory

Hit Command + K, and then Command + B. It would open or hide working directory.
